# 메뉴판결

오늘 뭐 먹을지 고민될 때 조건에 맞는 메뉴를 추천해주는 React + Vite 웹앱입니다.

## 주요 기능

- 음식 종류, 가격대, 기분, 혼밥 여부 필터
- 메뉴 판결 애니메이션
- 후보 메뉴 스크롤 목록
- 제외 메뉴와 최근 판결 기록 로컬 저장
- 검색 노출을 위한 메타태그, robots.txt, sitemap.xml 포함

## 실행

```bash
npm install
npm run dev
```

## 배포

`main` 브랜치에 push하면 GitHub Actions가 자동으로 빌드하고 GitHub Pages에 배포합니다.
